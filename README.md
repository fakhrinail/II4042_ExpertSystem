## Setup

1. Install virtualenv if you haven't

```
pip install virtualenv
```

2. Create virtualenv

```
virtualenv venv
```

3. Activate the created virtualenv

```
venv\Scripts\activate
```

4. Install dependencies

```
pip install -r requirements.txt
```

5. Export flask app

```
export FLASK_APP=app
```

6. Run flask

```
flask run
```
